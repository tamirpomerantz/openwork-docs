What is an API key and why does Openwork need it?

Openwork doesn't include its own AI model. Instead, it connects to an AI provider you choose, using an API key that belongs to you.

An API key is a private token that:

  • Identifies you to the AI provider
  • Lets Openwork send requests on your behalf
  • Bills usage directly to your account with that provider

Openwork uses your API key only to:

  • Generate text (summaries, drafts, explanations)
  • Understand files and instructions
  • Plan and execute actions you approve

Your key is stored locally on your machine and is never shared with Openwork servers.
